# Generative-AI

This repository contains my homework project for practicing a basic GitHub workflow with a coding assistant. It includes an assistant-generated draft, a final edited version file, and documentation of the steps I used during the assignment.

## Git Steps Used

I created a Git repository, added the required project files, committed my work as I progressed, connected the local repository to GitHub, and pushed the updates to the remote repository. I also merged the existing GitHub repository history with my local work before pushing.
